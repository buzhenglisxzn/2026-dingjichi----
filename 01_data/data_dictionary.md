## 2. 字段详细说明

### 2.1 Customer_ID（消费者ID）
- **数据类型：** 整数
- **范围：** 1 - 3900
- **唯一值数：** 3,900
- **含义：** 消费者的唯一标识符
- **缺失值：** 无

### 2.2 Age（年龄）
- **数据类型：** 整数
- **范围：** 18 - 80
- **平均值：** 45
- **含义：** 消费者的年龄
- **缺失值：** 无

### 2.3 Gender（性别）
- **数据类型：** 字符串
- **取值：** Male / Female
- **含义：** 消费者的性别
- **缺失值：** 无

### 2.4 Purchase_Amount（购买金额）
- **数据类型：** 浮点数
- **单位：** 美元 ($)
- **范围：** $10 - $500
- **平均值：** $150
- **含义：** 单次购买的总金额
- **缺失值：** 无

### 2.5 Frequency（购买频率）
- **数据类型：** 字符串
- **取值：** Weekly / Fortnightly / Monthly / Quarterly
- **含义：** 购买频率
- **缺失值：** 无

### 2.6 Category（商品类别）
- **数据类型：** 字符串
- **取值：** Clothing / Electronics / Home / Beauty
- **含义：** 购买商品的类别
- **缺失值：** 无

### 2.7 Season（季节）
- **数据类型：** 字符串
- **取值：** Spring / Summer / Fall / Winter
- **含义：** 购买时的季节
- **缺失值：** 无

### 2.8 Review_Rating（评价评分）
- **数据类型：** 浮点数
- **范围：** 1.0 - 5.0
- **平均值：** 3.5
- **含义：** 消费者对购买商品的评分
- **缺失值：** 无

### 2.9 Payment_Method（支付方式）
- **数据类型：** 字符串
- **取值：** Credit Card / Debit Card / PayPal / Bank Transfer
- **含义：** 支付方式
- **缺失值：** 无

### 2.10 Promo_Code_Used（是否使用促销码）
- **数据类型：** 布尔值
- **取值：** Yes / No
- **含义：** 是否使用了促销码
- **缺失值：** 无

### 2.11 Location（地区）
- **数据类型：** 字符串
- **取值：** Montana / Illinois / California / Texas 等
- **含义：** 消费者所在地区
- **缺失值：** 无

### 2.12 Satisfaction_Level（满意度）
- **数据类型：** 字符串
- **取值：** Low / Medium / High
- **含义：** 消费者满意度
- **缺失值：** 无
