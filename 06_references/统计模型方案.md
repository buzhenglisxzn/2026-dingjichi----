# 统计模型与分析方案

## 一、聚类分析方案

### 特征选择

选择以下特征进行聚类：
1. **Purchase_Amount**：购买金额（连续变量）
2. **Age**：年龄（连续变量）
3. **Review_Rating**：评价评分（连续变量）
4. **Frequency_Encoded**：购买频率（编码后的数值）

### 数据标准化

使用Z-score标准化，消除量纲影响：
$$x'_i = \frac{x_i - \bar{x}}{s}$$

### 最优K值确定

- 肘部法则：计算不同K值下的SSE
- 轮廓系数：选择轮廓系数最大的K值
- 预期K值：3-4

### K-means聚类

- 初始化方法：k-means++
- 最大迭代次数：300
- 运行次数：10
- 随机种子：42

## 二、假设检验方案

### T检验

**用途：** 比较不同年龄/性别的购买金额差异

**公式：**
$$t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}$$

**显著性水平：** α = 0.05

### 卡方检验

**用途：** 检验分类变量的关联性

**公式：**
$$\chi^2 = \sum_{i,j} \frac{(O_{ij} - E_{ij})^2}{E_{ij}}$$

### 方差分析（ANOVA）

**用途：** 比较多个组的均值差异

**公式：**
$$F = \frac{MS_{between}}{MS_{within}}$$
